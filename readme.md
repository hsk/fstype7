## リソースファイルの追加方法

1. ファイルをプロジェクトに追加する
2. ファイルを右クリックして [プロパティ] を開き [出力ディレクトリにコピーする] を選択する

	またはファイルを右クリック [クイックプロパティ] → [出力ディレクトリにコピーする] を選択する
